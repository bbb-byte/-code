## 章节命名规范

| 层级 | 格式 | 示例 |

| 节 | X.Y + 空格 + 标题 | 4.2 数据库设计 |
| 小节 | X.Y.Z + 空格 + 标题 | 4.2.1 ER图设计 |

## 各章节写作要点

### 第1章 绪论
- 研究背景：电商行业数据驱动运营的重要性
- 研究意义：用户画像对精准营销的价值
- 现状分析：简述RFM/聚类在电商领域的应用

### 第2章 相关技术
- RFM模型：R/F/M定义、五分位评分法
- K-Means：算法流程、K值选择（肘部法则）
- 漏斗分析：定义、序列约束

### 第3章 需求分析
- 功能需求：用例图、功能清单
- 非功能需求：性能、安全性、可扩展性

### 第4章 系统设计
- 架构图：前后端分离架构
- ER图：可直接使用 `ER_DIAGRAMS.md`
- 接口设计：RESTful API 列表

### 第5章 系统实现
- 核心代码片段（带注释）
- 截图展示（带标注）
- 实现细节说明

### 第6章 系统测试（重点）
- 测试用例表格
- 性能数据对比图
- 测试结论

### 第7章 总结与展望
- 工作总结：完成了什么
- 创新点：有什么亮点
- 不足与展望：后续可优化方向

## 格式要求提醒

| 项目 | 一般要求 |
|:---|:---|
| 字体 | 宋体/Times New Roman |
| 字号 | 正文小四号，标题三号加粗 |
| 行距 | 1.5倍行距 |
| 页边距 | 上下2.54cm，左右3.17cm |
| 图表 | 居中，标题在下方 |
| 代码 | 使用等宽字体（如Consolas） |

> **注意**：具体格式请以学校论文模板为准。
